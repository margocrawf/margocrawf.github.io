Guilt Motivator

Guilt Motivator is a to-do list app for Android with peer pressure incentives! Users add tasks that they need to get done, a due date, and one or more contacts, and if they don't check it off by the due date the app sends an email to their contact letting them know that the user has failed.

Technologies used include a SQLite database for contacts and tasks, background service, which tracks time and sends emails, and Java for basic in-app functionality. Some next steps include google login integration, texting, and desktop or iPhone versions, and an external database.
